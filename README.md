# 闪历 (Flash Calendar) - AI 智能工作计划工具

一个灵感源自 Apple 设计哲学的智能工作计划工具。闪历通过深度集成 Google Gemini API，将自然语言交互与流畅日历界面相结合，致力于通过 AI 穿透数据真相，提升个人生产力。

---

## 📸 视觉功能概览 (Visual Showcase)

> **注**：相关截图存放在项目的 `./assets/screenshots/` 文件夹下。

### 1. 核心看板 (Main Dashboard)
![主界面预览](./assets/screenshots/dashboard.png)
*   **设计语言**: 采用大面积留白、毛玻璃（Backdrop Blur）表头和 iOS 风格的圆角设计（24px+）。
*   **多维视图**: 侧边栏展示今日进度条与原子模版；中心区域为流畅的周日历视图，支持 `Cmd/Ctrl + 滚轮` 无级缩放。
*   **交互细节**: 每个日程方块根据内容高度自动切换显示模式（紧凑/详细），支持拖拽重排。

### 2. AI 实验室：镜像诊断 (Monthly Review)
![镜像诊断预览](./assets/screenshots/ai_diagnosis.png)
*   **视觉震撼**: 诊断弹窗采用暗色渐变与霓虹光晕效果，核心展示 AI 给出的**行为等级（S/A/B...）**。
*   **穿透真相**: 不同于普通的统计，AI 会分析你的“拖延模式”和“深度工作占比”，给出坦诚甚至有些“扎心”的行动建议。
*   **混沌扫描**: 背景中包含动态的“混沌扫描环”，视觉化展示你本月日程的混乱程度。

### 3. Flash Command 智能指令中心
![指令中心预览](./assets/screenshots/command_center.png)
*   **全局唤起**: 按下 `Cmd/Ctrl + K` 唤起类似 Raycast/Spotlight 的深色悬浮框。
*   **自然语言**: 输入“周五下午三点帮我约个技术评审”，AI 自动解析参数并完成排程预览。
*   **沉浸式交互**: 黑色半透明背景配合亮色文字，聚焦于指令输入，减少干扰。

### 4. 智能周报生成 (Weekly Report)
![周报预览](./assets/screenshots/weekly_report.png)
*   **一键汇总**: 自动提取本周已完成的任务，并根据进度推测下周计划。
*   **分段复制**: 针对钉钉、飞书优化的列表格式，支持一键复制到剪贴板。

## 🌟 核心特性

### 1. AI 驱动的自然语言交互
*   **语义化排程**: 只需输入“明天下午三点跟老板开会”，AI 即可自动识别时间、标题、并分配合理的起止时间。
*   **多模型兼容**: 默认支持 **Google Gemini 3 (Flash/Pro)**，同时支持 **DeepSeek (V3/R1)** 等任意 OpenAI 兼容接口。
*   **语音输入**: 集成 Web Speech API，配合动态音波视觉效果。

### 2. 极致的日历交互体验
*   **可视化周视图**: 支持缩放（60%-200%）的时间轴视图。
*   **拖拽式排程**: 支持从原子模版库拖拽任务块入日历，或直接在日历内拖拽调整任务时间。

### 3. 数据隐私与安全
*   **本地存储**: 采用 **IndexedDB** 数据库，所有日程数据均存储在您的本地浏览器中。
*   **备份管理**: 支持完整的 JSON 格式数据导入与导出。

## 🛠️ 技术栈

*   **前端框架**: React 19
*   **脚本语言**: TypeScript
*   **样式处理**: Tailwind CSS
*   **AI 引擎**: @google/genai (Gemini 3 系列)
*   **日期处理**: date-fns
*   **图标系统**: Lucide React
*   **数据存储**: IndexedDB

## ⌨️ 快捷键说明

| 快捷键 | 功能 |
| :--- | :--- |
| `Cmd/Ctrl + K` | 唤起/关闭 Flash Command AI 指令中心 |
| `Cmd/Ctrl + + / -` | 缩放日历视图 (Zoom In / Out) |
| `Cmd/Ctrl + 0` | 重置日历缩放级别 |
| `Esc` | 关闭当前弹窗或搜索框 |
| `Alt + 拖拽` | 快速复制并移动现有日程 |
